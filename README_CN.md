# blog
一个基于 [expressjs](https://expressjs.com/) ，使用 tailwindcss 作为前端的博客应用。

# 使用
## 快速开始
将本仓库克隆到你的文件夹中。
`$ git clone https://github.com/HelloLingC/blog.git`

安装 [Nodejs](https://nodejs.org/) (如果你还没有安装的话)。

通过下面的指令来下载并安装依赖。
`$ npm install`

将你自己的配置添加到 `/src/config/server.config.js`。

部署到拥有相同环境的服务器上。
